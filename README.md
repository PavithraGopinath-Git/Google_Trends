# Germany Migration & Student Search Trends Visualization

This project visualizes regional search interest related to migration and education pathways in Germany using data-driven mapping techniques.

## Overview

The visualization maps search interest for:
- Work visa information
- Student opportunities
- Migration-related queries

across German federal states. It demonstrates spatial data visualization using Plotly and geospatial mapping.

## Data

A sample dataset is used for demonstration purposes, representing relative interest levels across regions.

Columns:
- geoName – German state
- arbeitsvisum – Work visa interest index
- student – Student interest index
- migration_index – Combined average index

## Visualization

The choropleth map highlights:
- Regional variation in information demand
- Spatial patterns of migration/education interest
- Data-driven insights into public search behavior

## Technologies

- Python
- Plotly
- GeoJSON
- Pandas

## Purpose

This project is developed for academic data visualization and analytical demonstration. It showcases how digital search trends can be represented geographically.
